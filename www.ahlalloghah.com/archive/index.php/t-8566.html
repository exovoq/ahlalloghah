удЧхЯЩ ЧсфгЮЩ пЧусЩ : фцбё УжЧСЪ Шх ЧсЯфэЧ (дкб/ Эгф ЧсЭжбэ)

ЧумммммЯѕЯњ эммЯэмммпѓ смЭммШммс Чссх цЧкмЪѓммеіммуі = цЧгЪмлмнмбі Чссх уімммф Ьѕмммбњуђ цуімммф сѓмуѓммуі


нЧссх бШјѕмммммммп лмммнјѓммммЧбё смммуммммф оѓммнѓммсѓмммЪњ = Шмммх ЧсмбјіпммЧШѕ Хсмммь ЧсЪмгмШмэмЭ цЧсммфјѓммЯѓуі


хмммц Чсмммаэ ШмкммЫ ЧсумЮмЪмЧб дмЧнімкѓмфмЧ = нмдмојѓ ШЧсмфмцб уммЧ УкмэмЧ уімммф Чсмйјѕмсѓмуі



ЬммммммЧСЪ цсЧЯЪммммммх Шмммдмммбь цуѕмммфммммабЩ№ = цнммэ ЧсмЮмсЧЦмо пммму лмммЧцђ цукмЪѓмеімуі



гмујѓмЧх бШјѕммп нммэ ЧсмЪммцбЧЩ омммЯ кмсімумцЧ = уімф УумЩђ нжсѕхмЧ эгмумц кмсмь ЧсУуммуі



цЧЮмммммЪммммммЧбх Чссх Уумммммэј№ммммммЧ эммкммсјіммуѕммхммму = смму эмЪмсѕ уімммф пѕмЪѕммШђ Уц Юмммијѓ ШЧсмомсмуі



смму Ъмсмоѓ Ууммп Эмэмф Чсмцжмк ШмдјѓмбхмЧ = уумЧ ЪсЧомэ ЧсЭШЧсѓмь уімф Угммь ЧсУсммуі



цЧбЪммЯјѓЪі ЧсмЬмфјѕ Эмэмф ЧбЪммЧЯ уѕфњеіЪѕмхмЧ = Хсммь ЧсгмумЧС цаЧоммцЧ Чсммасјѓ уімммф Уѓуѓмммуі



цУэммоммфммцЧ Уф аЧп ЧсУуммммммб умфммпммдммнё = кф ЭЧЯЫђ Ьсѓсђ нэ ЧсУбж аэ кійѓуі



цЧхмЪммвјѓ ХэмммцЧф пмгммбь Эмэммф кмЧэмфммх = фммммцбѕ ЧсмфмШммэјі нммУбЬммЧх Хсммммь Чсмгјѓммоѓммуі



нммэ смЭмймЩђ дѓмхіммЯѓ ЧсмЪмЧбэмЮѕ бцкѓмЪѓмхмЧ = цумдмхмЯђ Уцоѓммммнѓ ЧсмЯфмэммЧ кммсммь оѓммммЯѓуі



фмммцбё УжмммЧСЪ Шмммх ЧсмЯфмэммЧ цШмдјѓмбѓхммЧ = Шммнммжммс бШјіммммммп ШмммЧсТэмммЧЪ цЧсммЭіммпѓмммуі



уіф уЭЪЯђ иЧхмб ЧсУфгмЧШ аэ дмбнђ = кмммммнјђ фммомммэјђ Ъммомммэјђ лммэмммб уммбЪммЬммуі



ЭѓммммШѓмммммЧх бШјѕммммммммммп Хф Чссх ац уімммммфѓмммммфђ = УумЧфммЩ ЧсмЯэммф цЧсмЯфмэммЧ Хсммммь Чсмфјѓмгѓммуі



цЧсммпммцф эмфмиммо ШЧсмЪмцЭмэмЯ эѕЪњмШімкѕмхмЧ = дмхммЧЯЩ№ смммп сммму Ъмдмимињ цсммму ЪѓмЮіммуі



эмммЧ емЧЯо№ммЧ эмммЧ Уумэмф№мЧ смэммг эѓмкњмЯісѕмх = нэ ЧсфмЧг уімф УЭмЯђ нмэ Ъісњпѕмуѕ Чсдјіэѓмуі



ХаЧ хмммммуѕ ЧфмеммбнммцЧ эммцу№мммЧ смШмЧимсмхму = УШѓммэњммЪѓ нммммэ кммммвЩ ЧсмЪмцЭмэмЯ пмЧсмкѓмсѓммуі



уЧ всмЪѓ нмэ ЧслмЧб ЪЯкмц Чссх укЪпн№мЧ = ЭЪмь хмЯЧпѓ Хсмь Чскймумь уммф Чсфјімкѓмуі



ЬѕммкіммсњммЪѓ ЮмммЧЪммму бѕгњммммммсі Чссх пммсммхмммуѕ = эммЧ Юмэмб уЮмЪѓмЪѓмуђ уімммф Юмэммб уЮмЪѓмЪімуі



омЯ кмЧхмЯцЧ Чссх эмцу№мЧ Эмэмф кЧхмЯхму = сѓмэмФуімфѕмфјѓ Шммммх нмммммэ гммЧЭмммЩ ЧсммЭѓмммбѓуі



ЯкмммцЪѓ оммцуммп ссмЪмцЭмэмЯ нмЧфмембнмцЧ = ХсЧ оммсммэммс№Ч ШммЭммоммЯђ лмммэмммб уммфммемммбуі



цоммсмммЪѓ Хфмммммэ бгммммммцс Чссх нмЧЪјѓмШіммкммцЧ = Яэмфммэ нмХфммэ фѓммеѕммцЭё лммэммб умпмЪѓмЪімуі



гмШмЭммЧф бШјіммммпѓ кммуммЧ оммммЧс омЧЦмсмхмму = цэмжмбШмцф смммп ЧсУумЫммЧс нммммэ Якѓммммуі



цЧссх бШјѕмммммммммммммпѓ кмммммммммммммсјѓЧуё ШммммУнммммЦмммммЯЩђ = жсЪ гШэс ЧсхѕЯѓь нэ гЧЭмЩ Чсйјѕсѓмуі



Хф эммЮммасмммцп нммЬѕммфмммЯѕ Чссх уѕммбњгѓмммсѓмммЩё = цЭѕмпњммуѕ бШјіммммп уммммЧжђ Юммэммб умЭмЪмпѓмуі



сммц дмммЧС бШјѕмммп сммму эмпмнмб Шмммх УЭмммЯё = сммпммфјѓ бШјѓммммп эммхммЯэ УЭммгмммф Чсмоіммэѓммуі



нмЧемШмб сммммх нмомжммЧС Чссх ац гѓммкѓммЩђ = цсмэммг умммф эмимсмШ ЧскѕмсњмэмЧ ШмумбЪмлмуі



омЯ пммЧф нммэ Уумму ЧсУгммсЧн уцкмймЩё = уімф омцу фммцЭђ Хсммь кэмгмь цнммэ Хбѓуі



хмму эумпмбцф цбШјѕ Чсмкмбд эухмсѕмхму = цЧсмЯхммб эмЬммбэ ШмЮммијі ЧсмсјѓммцњЭі цЧсмомсмуі
